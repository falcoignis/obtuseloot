# Trait Interference Fields Report

## Trait-pair categories
- trigger × mechanic
- mechanic × environment
- gate × trigger
- memory × mechanic
- branch tendency × environment affinity
- persistence style × support/combat role

## Amplification/dampening rules
- Bounded modifier range per evaluated ability: **[-0.18, +0.18]**.
- Trigger/mechanic synergies (movement echo chains, memory resonance chains) gain small amplification.
- Volatile low-health detonation pairings receive dampening.
- Environment affinity slightly shifts mechanic weights toward mobility/survival/chaos-aligned behavior.
- Interference outputs also emit bounded bias channels for mutation and latent activation.

## Emergent ability combinations
- Memory-event triggers + memory-echo mechanics can form delayed adaptation loops.
- Reposition/movement triggers + movement-echo mechanics form map-control pressure variants.
- Stable persistence + support modifiers produce resilient utility archetypes.

## Strategy diversity impact
The interference layer creates compact, deterministic score perturbations that increase combinatorial spread without causing score explosion, improving diversity while keeping seed reproducibility.
