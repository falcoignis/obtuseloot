# Ecosystem Balance Suggestions

## 1) Scope / sample size
- Inputs: isolated ecology reports + world-lab open-endedness ablation run.
- This document proposes conservative changes only, ordered by decision value.

## 2) Method summary
- Ranked issues by severity, confidence, and expected impact.
- Excluded broad architecture changes and speculative redesigns.

## 3) Key findings
- No critical generator-level collapse is present.
- Most meaningful risk is long-run concentration when ecosystem controls are weakened.
- Trait interactions appear important for preserving branch entropy.

## 4) Dominant families / branches / lineages / mechanics
- Dominant families and branches are stable and healthy in isolation.
- Dominant lineage recurrence across ablations indicates system remains partly designer-anchored.

## 5) Rare but viable systems
- Rare systems persist without hard extinction; this supports tune-small, observe-large strategy.

## 6) Dead or suspicious systems
- Suspicious: low turnover in some worlds and entropy collapse in no-trait-interaction mode.

## 7) Confidence / caveats
- Recommendation confidence:
  - Keep generator weights stable: **high confidence**.
  - Tune ecosystem controls conservatively: **moderate confidence**.
  - Retune trait interactions: **moderate confidence**.
  - Any large systemic conclusion: **provisional** until multi-seed reruns.

## 8) Actionable next review steps
1. **No immediate broad rebalance** (high confidence).
2. **Run 3-seed reruns for World A + C** before touching production balance constants.
3. If concentration rise reproduces, apply a **small** diversity-preservation uplift (5–10% band) and rerun.
4. If branch entropy in trait-interaction ablation continues to collapse, preserve/strengthen interaction influence rather than family weight inflation.
