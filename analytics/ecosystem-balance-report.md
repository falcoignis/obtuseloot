# Ecosystem Balance Report

## 1) Scope / sample size
- Isolated ecology sample: 5,000 artifacts.
- Companion evidence: world-lab ablation run (4 worlds, 80 players, 5 seasons, 12 sessions/season).

## 2) Method summary
- Read isolated generator distributions for families, branches, mutation outcomes, triggers, mechanics, and memory channels.
- Cross-checked with open-endedness world-lab outputs to separate generator balance from progression divergence.
- Applied confidence tags to avoid single-source conclusions.

## 3) Key findings
- Generator ecology remains broad (family Shannon 1.791, branch Shannon 2.889).
- No mutation occurs in 19.82% of sampled artifacts (watch-list, not immediate failure).
- World-lab confirms that divergence emerges mostly from ecosystem controls and trait-interaction dynamics rather than family template scarcity.

## 4) Dominant families / branches / lineages / mechanics
- Families: consistency (868), survival (866), precision (819), brutality (818).
- Branches: mobility.lane-dancer (300), chaos.paradox (300), chaos.awakened-entropy (296), mobility.relay (295).
- Mechanics: defensiveThreshold (409), chainEscalation (401), pulse (400), burstState (396).
- World-lab dominant trio was highly similar across worlds, indicating baseline generator pressure is stable but still somewhat designer-anchored.

## 5) Rare but viable systems
- precision.clock (251), mobility.fusion-slipstream (254), precision.awakened-discipline (257), survival.guardian (264).
- In world-lab, rare-lineage persistence remained non-zero across all worlds, indicating low-frequency systems can survive under pressure.

## 6) Dead or suspicious systems
- Dead memory types: none.
- Suspicious systems:
  - branch lock-in risk in trait-interaction ablation (World D entropy drop),
  - concentration growth when ecosystem controls are removed (World C).

## 7) Confidence / caveats
- Generator-level health conclusions → **high confidence**.
- "Mostly designer-controlled with measurable divergence" conclusion → **moderate confidence**.
- Long-run balance confidence remains **provisional** until multi-seed reruns are added for key ablations.

## 8) Actionable next review steps
1. Preserve current family generation mix; avoid broad generator rewrites.
2. Prioritize conservative tuning experiments around ecosystem control coefficients and novelty retention.
3. Run 3-seed reruns for World A and C before shipping medium/high-impact balancing changes.
