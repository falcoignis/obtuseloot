# Analytics Directory Guide

This folder stores text/JSON analysis outputs used for balancing and QA review.

- `evolution/` — evolution trend reports and data snapshots.
- `population/` — population-level simulation reports.
- `meta/` — cross-run meta-analysis and confidence summaries.
- `world-lab/` — world simulation lab reports and associated datasets.
- `review/` — review checklists, audit notes, and process docs.

Top-level analytics files are preserved for historical continuity where already referenced by scripts or docs.
