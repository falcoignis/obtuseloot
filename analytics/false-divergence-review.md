# False Divergence Review

False divergence in this repository means species and niche activity appears high, but persistent ecological differentiation does not improve.

## Signals used

- high species count with weak divergence
- high niche activity with low END
- high TNT with low sustained NSER
- relabeling/migration without persistence
- novelty signatures that fail to persist
- dominant attractor concentration remaining high

## Current run assessment

Current fused diagnostic state is **STAGNANT_ATTRACTOR** (not false divergence), because END remains low and the ecology still collapses into a concentrated attractor. False-divergence flag is currently not primary; stagnation remains the stronger signal.
