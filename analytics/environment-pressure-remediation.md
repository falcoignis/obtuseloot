# Environment Pressure Remediation

## What changed
- Extended environmental event duration from 1–3 seasons to 2–5 seasons (longer cycles).
- Broadened event modifier coverage to include resonance/stability/survival/mutation-sensitivity dimensions so pressure influences non-combat pathways too.
- Tightened environment gate coupling so environmental state has stronger eligibility impact.

## Expected ecosystem impact
- Longer cycles provide sustained selective windows rather than short spikes.
- Broader influence opens additional niches beyond direct combat optimization.
- Stronger gate coupling increases context-sensitive branch expression.

## Risk level
- **Overall risk:** medium-low.
- Temporary over-favoring of event-aligned lineages is possible but bounded by capped modifiers and gate thresholds.
