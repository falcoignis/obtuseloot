# Novel Strategy Emergence Report (NSER)

## Strategy signature definition
Signature = [type (artifact/species), niche identity, dominant regulatory gate profile, dominant trigger class, dominant mechanic class, branch role/preference, co-evolution tendency].

## Significance rules
A signature is significant when observation count >= 3, occupancy share >= 0.05, and persistence >= 2 seasons. Novelty additionally requires max similarity < 0.75 vs prior significant signatures.

## NSER thresholds
- NSER < 0.05 => no real novelty
- 0.05 <= NSER < 0.15 => weak novelty
- 0.15 <= NSER < 0.30 => healthy novelty
- 0.30 <= NSER < 0.50 => strong innovation phase
- NSER >= 0.50 => likely over-fragmented/noisy
