# ObtuseLoot v0.9.5 Changelog

## Gameplay + progression systems through 0.9.5
- Expanded procedural ability generation coverage so artifacts can express richer family/branch outcomes tied to combat behavior patterns.
- Continued mutation system tuning and reporting so drift/mutation outcomes remain observable and balanceable.
- Kept ability evolution tree pathing and branch-resolution workflows as first-class progression mechanics.
- Preserved and validated artifact memory influence as a core progression signal across evolution, mutation, awakening, and fusion touchpoints.

## Analytics + balancing support
- Maintained analytics outputs for ecosystem health, population behavior, and world-lab simulation review.
- Kept balancing AI style outputs and meta-analysis artifacts available for iterative design decisions.

## Repository hygiene for this release line
- Removed committed Maven build output from version control and formalized ignore rules for local build/artifact paths.
- Added dedicated release docs for v0.9.5 (changelog, QA notes, build info) without committing binaries.
